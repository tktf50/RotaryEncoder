Detailed Description

template<unsigned char DtPin, unsigned char ClkPin>
class RotaryEncoder< DtPin, ClkPin >

Rotary encoder interface. Provides indication of encoder rotation and direction of rotation.

Based on the original work of Ben Buxton (Copyright 2011) and released under the GNU GPL Version 3.

Template Parameters
DtPinGPIO connected to the encoder DT pin.
ClkPinGPIO connected to the encoder CLK pin.

◆ getAndResetEncoderDir()

template<unsigned char DtPin, unsigned char ClkPin>
unsigned char RotaryEncoder< DtPin, ClkPin >::getAndResetEncoderDir ( )
inline

For interrupt driven processing, get the most recent encoder turn direction and reset it to DIR_NONE. Therefore, calling this method a second time before another turn has occurred will return DIR_NONE.

You can also access the encoder direction directly if you require more complex processing of turn events.

Returns
unsigned char One of DIR_NONE, DIR_CW or DIR_CCW
